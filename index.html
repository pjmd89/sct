Formularios

El elemento padre de los elementos de formularios debe contener la clase "sct-control"

"sct-control" tiene definida 4 helpers:

  • sct-control-align-left: Establece el width de los label a 160px y alinea el texo de este mismo a la izquierda
  • sct-control-align-right: Establece el width de los label a 160px y alinea el texo de este mismo a la derecha
  • sct-control-inline: Une a los elementos que esten dentro del mismo padre "sct-control-inline". Tenga en cuenta que este helper NO HACE USO DEL ATRIBUTO FLOAT, por este motivo debe suprimir los espacios entre las etiquetas inline
  • sct-control-textarea-expand: Aumenta el tamaño del texarea colocandolo por debajo del label
